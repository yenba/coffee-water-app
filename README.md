# Coffee Water App

**Direct Dosing Recipe Picker & Calculator** for nerds who build their own coffee water.

This tool helps you calculate how many minerals to add directly into water to achieve specific water hardness (GH) and alkalinity (KH) levels for brewing coffee. All calculations happen instantly in your browser — no data is sent anywhere.

## Features

- **Recipe Picker**: Pick a well-known water recipe from the dropdown and choose your desired water amount. The calculator tells you exactly how many grams of each mineral to add.
- **Recipe Builder**: Create your own recipe by entering your desired GH and KH values. The calculator figures out how much of each mineral you need.
- **GH/KH Lookup**: The reverse calculator — enter the grams you used and it tells you the resulting GH and KH of your water.

## Development

This is a React application built with TypeScript and Vite. It uses Tailwind CSS for styling.

### Setup

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build
```
